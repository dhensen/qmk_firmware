# DZ60 Vulcan Keymap

Default: macOS (Cmd next to space). Toggle L1 for Windows/Linux.
Tap-hold keys shown as: Tap / Hold

## Quick Reference

| Action             | Keys                                                |
| ------------------ | --------------------------------------------------- |
| Function keys      | Hold Down + number row                              |
| Mouse mode         | Hold Esc or Hold Left arrow                         |
| Toggle Win/Linux   | Hold Esc + Tab                                      |
| Volume             | Hold Down + , . /                                   |
| RGB controls       | Hold Down + Q W E R T Y U I                         |
| Backlight          | Hold Down + C V B N                                 |
| Bootloader (flash) | Hold Down + \                                       |
| Print Screen       | Hold Esc + P                                        |
| Page Up/Down       | Hold Esc + Up / Hold Esc + Down (bottom-right)      |
| Home/End           | Hold Left + Left / Hold Left + Right (bottom-right) |
